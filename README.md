# 🛒 SmartShoppingKeep

スマートフォンでの利用に特化した、買い出し時の金額超過を防ぐ「リアルタイム計算機能付き」買い物メモアプリです。
スーパーやドラッグストアなどの売り場を想定し、片手でサクサク操作できるデザインと、10%・8%（軽減税率）・免税が混在しても1円単位で正確に自動集計するプロ仕様のロジックを搭載しています。

## 🚀 デプロイ先（Web公開URL）

## 🌟 主な機能
- **購入予定場所（カード）管理**: 「スーパー」「コンビニ」など、場所ごとにメモをカード型で管理・丸ごと削除可能。
- **スマートフォン最適化（レスポンシブデザイン）**: 片手でもチェックや金額入力、削除がしやすい直感的なUI。
- **非同期リアルタイム金額集計 (Fetch API)**: カゴに入れた瞬間(チェックボタンクリック時)に金額入力欄が開き、数値を打ち込むと「税抜・各税率ごとの内訳・消費税・税込合計」がその場で即座に再計算・自動保存されます。
- **複数税率（標準/軽減/免税）対応**: アイテムごとに「10%」「8%」「免税」をワンタップで切り替え可能。端数（切り捨て）も実店舗のレジと同一の計算ロジックを再現。
- **サクサク削除機能**: 不要になったアイテムや、用済みのメモを画面遷移なしでシュッと消去可能。

- **フロントエンド**: HTML5 / CSS3 / JavaScript (Vanilla JS) / Bootstrap 5
- **デザイン対応**: レスポンシブ対応（モバイルファースト）
- **ソースコード管理**: Git / GitHub

## 💡 こだわったポイント
1. **ユーザー体験（UX）の向上**: 実際の買い物中にパソコンを開く人はいないため、初期設計からブラッシュアップし、完全なモバイル向けUIへとシフトしました。
2. **実用的な消費税ロジック**: お惣菜（8%）と日用品（10%）を同時にカゴに入れても、裏側と表側のJavaScriptがそれぞれを分けて自動集計し、実店舗のレジと完全に一致する税込合計を算出するように作り込みました。
